This project implements a Linear Regression model to predict house prices using the Kaggle House Prices dataset.

The model uses key features such as square footage, number of bedrooms, and number of bathrooms to estimate property prices.

Key Highlights:
- Data preprocessing and feature selection
- Model training using Linear Regression
- Evaluation using R² score
- Visualization of actual vs predicted prices

Tools & Technologies:
- Python
- Pandas
- Scikit-learn
- Matplotlib

The project demonstrates the fundamentals of supervised learning and regression analysis.
